Situation & Task

Creating a simple, reliable, and cost-effective security system is a common electronics challenge. The objective of this project was to design and build a sensitive intruder alarm from basic analog components that could detect movement and provide an immediate, audible notification without the need for a microcontroller.

My Role & Responsibilities:

I was the sole designer and engineer for this project. My responsibilities included designing the complete analog circuit, selecting the appropriate ICs and passive components, creating the circuit diagram, and building and testing the physical prototype on a breadboard, including waveform analysis with an oscilloscope.

Action

I designed a multi-stage analog circuit to process the sensor signal and trigger the alarm:

1. IR Sensor Detection

An IR sensor module was used as the primary motion detector. When an object obstructs the sensor, its digital output pin goes from HIGH to LOW.

2. Signal Inversion & Comparison

The LOW signal from the sensor is fed into an LM358 op-amp. I configured the LM358 as a comparator to compare the sensor's output to a reference voltage. This stage cleans the signal and produces a reliable HIGH output when an intruder is detected.

3. Alarm Generation with 555 Timer

The HIGH output from the LM358 triggers the reset pin of an NE555 timer IC. I configured the 555 timer in an astable multivibrator mode, which generates a continuous square wave. This wave drives a buzzer, producing a loud, audible alarm tone.

Result

The project resulted in a fully functional and highly responsive intruder alarm prototype. The circuit performed exactly as designed, demonstrating a successful application of analog electronics principles.

Reliable Detection

The final breadboard prototype successfully and instantly triggered the buzzer and an indicator LED whenever an object was detected by the IR sensor.

Waveform Validation

Using an oscilloscope, I verified that the NE555 timer was producing the correct square wave frequency as calculated, confirming the astable circuit was operating correctly.

Key Learnings

  • This project solidified my understanding of fundamental analog components, particularly the practical application of an LM358 op-amp as a comparator and the NE555 timer in astable mode.
  • I gained valuable hands-on experience in breadboard prototyping, which involves careful component placement and wiring to minimize noise and ensure a stable circuit.
  • Using an oscilloscope to test and debug the circuit was a key learning experience, reinforcing the importance of using proper test equipment to validate that a physical circuit's performance matches theoretical calculations.
